# TypeScript Boilerplate

A minimal TypeScript starter template with modern tooling.

## Stack

- Node.js 20+
- TypeScript 5.8
- Jest & Vitest for testing
- ESLint + Prettier

## Getting Started

Install dependencies:

```sh
npm install
```

Run in development mode:

```sh
npm run dev
```

Build for production:

```sh
npm run build
```

Run the built application:

```sh
npm start
```

Run tests:

```sh
npm test                # Jest
npm run test:vitest     # Vitest
```

Format code:

```sh
npm run format          # Format all files
npm run format:check    # Check formatting
```

Lint code:

```sh
npm run lint
```
